# Personal Homepage

A lightweight academic personal homepage inspired by the Minimal Light style.

## Edit Content

Update the data arrays at the top of `assets/site.js`:

- `profile`: name, affiliation, email, avatar, external links
- `about`: short bio paragraphs
- `interests`: research interests
- `news`: recent updates
- `publications`: paper list and links
- `projects`: selected projects

## Preview

Open `index.html` in a browser, or serve this directory with any static server.

## Deploy

For GitHub Pages, push these files to a repository and enable Pages from the repository settings.
